# ARN

ARN（Active Render Network）是一种基于强化学习的连续视觉自编码器框架。它不再把图像表示视为固定长度的 patch 集合，而是把重建过程建模为一个序贯决策过程：模型在时间维度上持续观察目标图像与当前画布，并逐步生成一串连续动作向量，最终完成从粗到细的高保真重建。

## 项目理念

ARN 由两个核心模块构成：`Actor` 负责生成连续动作序列，作为图像的动态隐表示；`Renderer` 负责将这些动作逐步解码为画布残差，最终逼近目标图像。与基于人工规则的图形学渲染不同，ARN 希望通过纯数据驱动的神经渲染方式，直接学习从连续特征到像素分布的映射，尽可能减少风格化偏差与手工先验的限制。

这套设计追求三件事：一是变长连续编码，让简单图像用更短序列、复杂图像用更长序列表达；二是全局空间建模，使每一步动作都面向整幅图像而非受限于光栅扫描；三是把图像压缩、视觉 token 化与序列建模统一到同一个自回归式框架中。训练完成后，`Actor` 不只是重建器的一部分，也会成为一个可用于压缩、生成、编辑和多模态建模的动态视觉 Tokenizer。
